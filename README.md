# **Tittle:**

YouTube NLP Summarizer

#### Description:

This project extracts transcripts from YouTube videos and generates summaries using Natural Language Processing techniques.

#### Features:

* Extract transcript from YouTube videos
* Summarize text using NLP models
* Multi-language support
* Streamlit-based UI

#### Tech stack:

Technologies Used

* Python
* Streamlit
* Transformers (HuggingFace)
* Whisper / YouTube Transcript API

#### Installation:

1\. Clone the repository:

&#x20;  git clone https://github.com/your-username/your-repo.git



2\. Navigate to project folder:

&#x20;  cd youtube-nlp-summarizer



3\. Install dependencies:

&#x20;  pip install -r requirements.txt

#### How to Run:

Run the Streamlit app:

streamlit run app.py
